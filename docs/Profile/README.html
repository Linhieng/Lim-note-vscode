Profiles 的作用是为不同的项目使用不同的配置项。

全局扩展 ​

全局扩展，表示开启了“Apply Extensions to all Profiles”。

只将我经常使用的扩展和每个项目都会用到的扩展列入全局扩展，这意味着这些扩展不会专门添加到项目的 .vscode/extensions.json。
